# Парсер книг с сайта [tululu.org](https://tululu.org)


Данный проект предназначен для скачивания книг из большой библиотеки. Проект также помогает загрузить вместе с книгой:

1. Автора книги
2. Комментарии пользователей
3. Жанр книги
4. Обложку книги

## Как установить

Python3 должен быть уже установлен. Затем используйте pip (или pip3, если есть конфликт с Python2) для установки зависимостей:

```pip install -r requirements.txt```

## Как запустить скрипт

Для того, чтобы скачать книги, необходимо прописать команду в терминал:

```python main.py "указать диапозон книг"```

Например:
```python main.py 20 30```
- данная команда произведет загрузку 10 книг с id книги от 20 до 30 включительно.
