# Парсер книг с сайта [tululu.org](https://tululu.org)


Данный проект предназначен для скачивания книг из большой библиотеки. Проект также помогает загрузить вместе с книгой:

1. Автора книги
2. Комментарии пользователей
3. Жанр книги
4. Обложку книги

## Как установить

Python3 должен быть уже установлен. Затем используйте pip (или pip3, если есть конфликт с Python2) для установки зависимостей:

```pip install -r requirements.txt```

## Как запустить скрипт

Для того, чтобы скачать книги, необходимо прописать команду в терминал:

```python parse_tululu_book.py "указать диапозон книг"```

Например:
```python parse_tululu_book.py --start_id 20 --end_id 30```
- данная команда произведет загрузку 10 книг с id книги от 20 до 30 включительно.

Если необходимо скачать сразу много страниц с книгами:

```python parse_tululu_category.py "указать диапозон страниц"```

Например:

```python parse_tululu_category.py --start_page 20 --endpage 30```
- данная команда произведет загрузку книг со страницы 20 до 29 включительно.
- если не указать параметр --endpage, то книги загрузятся с 20 и по последнюю страницу(701)

Есть дополнительные параметры, которые можно использовать с командами выше:
- --skip_imgs позволяет скачать все, кроме картинок
- --skip_txt позволяет не скачивать текст книги, а только основную информацию о ней
- --dest_folder при указании данной команды указывается путь, куда вы хотите скачать файлы

Например:

```python parse_tululu_category.py --start_page 20 --endpage 30 --skip_imgs```
команда скачать все книги с 20 по 29 страницу, но не загрузит картинки для них

```python parse_tululu_category.py --start_page 20 --endpage 30 --skip_txt```
команда скачать всю информацию о книгах с 20 по 29 страницу, но сам текст скачан не будет

```python parse_tululu_category.py --start_page 20 --endpage 30 --dest_folder boooks/```
команда скачать все книги с 20 по 29 страницу и загрузить их в указанную вами папку boooks(default=books)

Такие же параметры можно использовать и с файлом main для этого вместо parse_tululu_category.py прописать main.py

Например:

```python parse_tululu_book.py --start_page 20 --endpage 30 --skip_imgs```
